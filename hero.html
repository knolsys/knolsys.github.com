About

In a very short sentence, our expertise is on developing data management and processing systems to deal with heterogeneous big data, relying on scalable and efficient tools and infrastructures, and using semantic Web technologies that enable to link the data in a way that the structure that represents the knowledge emerges from the data.

Main action lines: consultancy; systems Development; data Analysis.
